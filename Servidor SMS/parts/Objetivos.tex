\section*{Objetivo General}
Poner en funcionamiento el desarrollo en equipo de un tel\'efono IP.

\section*{Objetivos Específicos}
\begin{itemize}
	\item Estudiar las caracter\'isticas del hardware para desarrollar el tel\'efono IP.
	\item Desarrollar el Hardware y el Software para el prototipo de la interfaz humana.
	\item Analizar la disponibilidad en el mercado de los elementos de hardware para el desarrollo de la interfaz humana.
	\item Dise\~nar la placa de la interfaz humana.
	\item Dise\~nar un protocolo para la comunicaci\'on entre la interfaz humana y el m\'odulo de red.
	\item Ensamblar la interfaz humana con el m\'odulo de red.
\end{itemize}
